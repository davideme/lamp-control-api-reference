# Java Implementation

This directory contains the Java implementation of the Lamp Control API.

## Features

- REST API (Spring Boot with OpenAPI 3.0)
- GraphQL (Spring GraphQL)
- gRPC
- Database support for MySQL, PostgreSQL, and MongoDB
