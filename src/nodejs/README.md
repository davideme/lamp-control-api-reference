# Node.js Implementation

This directory contains the Node.js implementation of the Lamp Control API.

## Features

- REST API (Express + OpenAPI 3.0)
- GraphQL (Apollo Server)
- gRPC
- Database support for MySQL, PostgreSQL, and MongoDB
